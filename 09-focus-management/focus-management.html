Focus Management

For focusable elements there are four states / scenarios:

  • By default, active and focus are both off
  • When you tab to cycle through focusable elements, they will enter :focus (without active)
  • When you click on a non-focusable element, it enters :active (without focus)
  • When you click on a focusable element it enters :active:focus (active and focus simultaneously)

Using "What Input?"

What Input uses event bubbling on the window to watch for mouse, keyboard and touch events (via mousedown, keydown and touchstart). It then sets or updates a data-whatinput attribute.

Used with the following CSS, the focus ring is removed for mouse users

/* Suppress focus ring on form controls for mouse users */
[data-whatintent='mouse'] *:focus {
  outline: none;
}
Web accessibility summary
When websites and web tools are properly designed and coded, people with disabilities can use them. However, currently many sites and tools are developed with accessibility barriers that make them difficult or impossible for some people to use.
